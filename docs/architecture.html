Auth flow (JWT Bearer)

1.POST /auth/login → token JWT
2.Token stocké localStorage
3.Header Authorization sur chaque req
4.Middleware auth.go vérifie la signature
5.Expiration : 480 minutes
6.401 → logout auto + redirect /login
